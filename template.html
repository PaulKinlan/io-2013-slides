Viewport

The most critical element for mobile TODAY!!

  • initial-scale
  • minimum-scale=0.1-1.0
  • maximum-scale=0.1-1.0

Viewport queries

A saner solution
@viewport {
  width: device-width;
  zoom: 0.5;
}
  • Great delivery flexibility
  • Increased readability

Media Queries

@media screen and (min-device-width: 800px) { ... }
@media screen and (device-aspect-ratio: 16/9) { ... }
@media screen and (min-device-pixel-ratio: 3/2) { ... }

Flex Box

.flex {
  display: flex;
  flex-direction: row | row-reverse | column | column-reverse;
}
  • Combines well with Media queries
