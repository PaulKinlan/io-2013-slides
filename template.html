Viewport

The most critical element for mobile TODAY!

<meta name="viewport" content="" />

Device width

<meta name="viewport" content="width=device-width" />

Initial Scale

<meta name="viewport" content="initial-scale=0.5" />

Minimum Scale

<meta name="viewport" content="minimum-scale=0.5" />

Maximum Scale

<meta name="viewport" content="maximum-scale=0.5" />

Viewport queries

A saner solution

@viewport {
  width: device-width;
  zoom: 0.5;
}
  • Great delivery flexibility
  • Increased readability

Viewport units

Sizing relatively to the viewport

#visible-square {
  width: 100vmin;
  height: 100vmin;
}
  • Four new units: vw(idth), vh(eight), vmin, vmax
  • Relative to the size of the viewport

Media Queries

Adaptive to screen size

@media screen and (min-device-width: 800px) { ... }
@media screen and (device-aspect-ratio: 16/9) { ... }
@media screen and (min-device-pixel-ratio: 3/2) { ... }
@media screen and (resolution: 2dppx) { ... }
  • Logically group css
  • Dynamically responsive

Flex Box

Adaptive layouts

#id {
  display: flex;
  flex-direction: row | row-reverse | column |
      column-reverse;
}
  • Combines well with Media Queries
  • Adaptive to display size

Spec: http://dev.w3.org/csswg/css-flexbox/

Responsive Images

image-set

#image1 {
  content: -webkit-image-set("1x test.png, 2x test2.png");
}
  • Combines well with Media Queries
  • Adaptive to display size
